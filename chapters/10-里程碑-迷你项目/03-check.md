> 这一天是里程碑，不出题目，而是用清单做自我评估。

## 项目完整性检查

- [ ] 项目可以从零 `node src/index.js` 启动，没有报错
- [ ] `GET /todos` 返回空数组（无数据时）
- [ ] `POST /todos` 创建成功，返回 201 和新对象
- [ ] `GET /todos/:id` 返回单个，ID 不存在返回 404
- [ ] `PATCH /todos/:id` 只更新传入的字段
- [ ] `DELETE /todos/:id` 返回 204，再 GET 该 ID 返回 404
- [ ] `GET /todos?page=1&limit=5` 返回分页数据
- [ ] 不传 title 创建时返回 400

## 代码质量检查

- [ ] 没有所有代码堆在一个文件（至少分了 router、middleware、routes）
- [ ] 自定义 Error 类（至少有 NotFoundError、ValidationError）
- [ ] 错误处理中间件兜底（不会 500 时暴露 stack trace）
- [ ] README.md 写清楚了如何启动和 API 列表
