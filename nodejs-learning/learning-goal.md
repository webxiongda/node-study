# 学习目标

## 学习动机
面试导向：目标是通过全栈岗位面试（含大厂）

## 当前基础
有一些了解，不系统（非零基础）

## 可投入时间
每天约 2 小时

## 验收标准
1. 能独立编写 REST API（含认证、数据库、缓存）
2. 能通过全栈岗位面试

## AI 推荐重点

### 必须深度掌握（直接影响面试）
- Node.js 运行时原理（事件循环、libuv、异步 I/O）
- 模块系统（CJS vs ESM）
- Promise / async-await / 并发控制
- HTTP 基础 + 手写 REST API（理解框架底层）
- TypeScript 服务端开发
- NestJS 架构（IoC/DI、模块/控制器/服务、请求生命周期）
- JWT 认证 + RBAC 权限
- PostgreSQL + Prisma ORM
- Redis 缓存策略
- 系统设计基础（CAP、负载均衡）

### 项目实战（简历必备）
- 手写迷你 TODO API（Day 8-10）
- 博客系统后端（NestJS + PostgreSQL + Redis，Day 16-40）
- SaaS 任务管理平台（全栈，Day 46-55）

### 可适当快过（了解即可）
- 环境搭建细节（你已有基础，快速过）
- Docker/CI/CD（了解概念，不需要精通）
- 测试（Supertest 了解即可，精力优先给核心）
