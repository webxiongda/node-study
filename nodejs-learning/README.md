## 优先级说明
- **L1 必须掌握**：面试 + 项目都要，不能跳过。
- **L2 项目常用**：工作中经常用，建议掌握真实业务边界。
- **L3 面试高频**：面试追问概率高，要准备结构化表达。
- **L4 进阶拓展**：工程化和加分项，有余力时深入。

## 学习方法
1. 每章先读 `01-theory.md`，只抓核心概念和应用场景。
2. 再做 `02-demo.md`，必须跑出可验证结果。
3. 学完后完成 `03-check.md`，用自己的话讲清楚。
4. 最后做 `04-project-task.md`，把知识落到真实项目。
5. 通过验收后再看 `review.md`，并把错题写入 `mistakes.md`。
