# 推荐阅读资源

## 📚 书籍

### 必读
1. **《Designing Data-Intensive Applications》** (Martin Kleppmann)
   - 系统设计圣经，重点读第 1-9 章
   - 涵盖数据存储、分布式系统、一致性、分区等核心概念

2. **《SQL Antipatterns》** (Bill Karwin)
   - 避免数据库设计中的 25 种常见错误
   - 每种反模式都有清晰的问题描述和解决方案

### 推荐
3. **《Web Scalability for Startup Engineers》** (Artur Ejsmont)
   - 系统扩展性入门，适合从零理解分布式系统

4. **《Node.js Design Patterns》** (Mario Casciaro)
   - Node.js 设计模式经典，深入理解架构模式

5. **《Clean Code》** (Robert C. Martin)
   - 代码质量圣经，适用于所有语言
