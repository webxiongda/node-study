## 💻 限时编码挑战

### 挑战 1：HTTP 文件服务器（30 分钟）

用原生 Node.js 实现一个文件浏览和下载服务器：
- 访问目录时显示文件列表（HTML 页面）
- 点击文件名可以下载
- 正确的 Content-Type
- 支持子目录浏览

### 挑战 2：并发爬虫（30 分钟）

编写一个简易爬虫，给定一组 URL：
- 并发抓取（最大并发数 5）
- 提取每个页面的 `<title>`
- 支持超时（10 秒）
- 统计成功/失败数量
- 将结果保存为 JSON 文件

### 挑战 3：实时日志监控（20 分钟）

实现一个实时日志监控工具：
- 监控指定的日志文件
- 新增的行实时输出到终端
- 高亮 ERROR 级别的日志（红色）
- 统计各级别日志数量
